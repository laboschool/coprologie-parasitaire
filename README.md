Version autonome pour GitHub Pages.

Contenu:
- `index.html`: version self-contained du manuel avec la section `Commentaires typiques` integree.
- `.nojekyll`: evite tout traitement Jekyll inutile.

Usage recommande:
1. pousser le contenu du dossier `github-pages` dans le depot ou dans la branche de publication,
2. configurer GitHub Pages pour publier ce dossier si besoin,
3. verifier que l'URL finale pointe bien vers `.../github-pages/` si ce sous-dossier est conserve tel quel dans le site.
